PA9 - GRAPH TEST APPLICATION
============================

Project description:
This program is a graph implementation test program. The program continuously 
repeatedly prompts the user to operations to perform on graph structure created 
form the file AdjListFile.txt.


How to compile and run:
Create an empty C++ makefile project and import the attached files into the project and build the
application with the MinGW GCC toolchain or any other compiler and check that the binary files are create. 
Please check that you add the commandline argument -std=c++0x to your build settings.
You can then run it from the command line.

A sample run and the graph file are attached.
