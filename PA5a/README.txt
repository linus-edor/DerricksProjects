PA5 - LINKED LISTS
===================

Project description 
-------------------
This program is a commandline program that presents a menu of list operations to perform on an unordered linked list.
The program continually prompts the use for an option and performs the chosen operation on the list object.
The user also has an option to quit the application.


Execution guidelines
--------------------
How to compile and run:
Create an empty C++ makefile project and import the attached files into the project and build the application 
with the MinGW GCC toolchain or any other compiler. Please check that you add the commandline argument -std=c++0x.
